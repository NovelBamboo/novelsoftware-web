# 💸 NSD — Pricing & Go-To-Market

## Pricing Model

Hybrid Consulting Model

Separate pricing for:

- Discovery / Diagnostic
    
- System Implementation
    
- Ongoing Knowledge Operations
    

Pricing is based on:

- Repository complexity
    
- Number of workflows
    
- Migration effort
    
- Governance requirements
    
- Organizational size
    

Not:

- Hours worked
    
- Number of documents
    
- AI usage
